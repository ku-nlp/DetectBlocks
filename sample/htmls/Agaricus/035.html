免疫活性効果に優れたきのこ

アガリクス

すこやかな毎日を過ごす為に

カバノアナタケ ( チャーガ )、 ヤマブシタケ、ハナビラタケ、メシマコブ等々免疫促進効果に優れたきのこが最近話題を集めていて、免疫力 ( 自然治癒力 ) が高ければ病気にかかりにくい又は病気に対する抵抗力が強いと考えられてます。免疫力は食生活ストレス過労生活環境加齢等様々な要因によって徐々に低下していきます。この免疫力を高める事が健康を維持するのにすごく重要な事になります。


アガリクスはどのようなきのこ？

アガリクス(agaricus)はハラタケ科ハラタケ属の担子菌類のきのこの総称でラテン語できのこという意味です。
お馴染みのマッシュルーム(シャンピニオン)もハラタケ科ハラタケ属の担子菌類のきのこで学名アガリクス・ビスポラス(Agaricus Bisporus)と言います。
健康食品として脚光を浴びている通称［アガリクス］は学名アガリクス・ブラゼイ・ムリル(Agaricus Blazei Murill)と言い、アガリクス(アガリクス・ブラゼイ・ムリル)の原産地はその名が示す通りブラジルサンパウロ郊外のピエダーテ地方の山岳地方です。この地方は土地が肥沃で昼夜の気温差が激しく湿度が高く、また定期的にスコールが訪れます。アガリクスはこのような限られた気象条件下でしか自生しません。そしてこのような気象条件を備えるのはピエダーテ地方だけです。

アガリクスが注目されるようになったキッカケは1965年ペンシルバニア大学のW.J.シンデン博士とランバート研究所のE.B.ランバート博士を中心とした研究チームによる疫学調査発表でした。ピエダーテ地方には生活習慣病の罹患者が少ない事からその原因を調査した結果同地方の住民がアガリクスを常食している事更にアガリクスがビタミン・ミネラル核酸酵素等の成分を沢山含有している事も突き止めました。そして同年この結果が学会誌に発表され日本にもアガリクスが持ち込まれました。
アガリクスはメシマコブと同様にに人工栽培が難しいきのこでしたが現在では子実体や菌糸体の人工栽培に成功し市場に安定供給されてます。
アガリクスもまた免疫力を高めるきのことして脚光を浴びてます。

アガリクスどのような効果や効能があるの？

アガリクスはたんぱく質、多糖類、ビタミン(B2 ， D) ミネラル ( マグネシウム，カリウム ) 脂質 ( リノール酸やパルミチン酸 ) 食物繊維、核酸、アミノ酸、必須脂肪酸等を沢山バランスよく含んでます。

特に高分子多糖体の一種β -D- グルカンの含有量はきのこの中で最も多いと言われてます。このβ -D- グルカンは子実体 ( 通常、きのこと認識している部分 ) よりも菌糸体 ( 子実体の根元のような部分から土壌の中又は動植物の中に伸びていく白い絹糸状のもの ) に多く含まれてます。β -D- グルカンはグリコプロテインとも呼ばれたんぱく質と多糖体がバランスよく結合する事によってできる糖たんぱく質です。

β -D- グルカンは免疫力のバランス感覚である生体調整機能を改善する効果がありアガリクスは特にその効果が高いと考えられてます。これによりアガリクスは身体が本来持っている自然治癒力を高める為あらゆる病気に有効であると言われてます。中でも免疫力の低下によって引き起こされる生活習慣病 ( ガン糖尿病高脂血症肝臓疾患等 ) や逆に免疫力の過剰活性によって引き起こされるアレルギー性疾患 ( アトピー性皮膚炎花粉症等 ) に効果を発揮すると期待されてます。

アガリクスについては様々な研究がされてます。特に抗ガン活性 ( 抗腫瘍活性 ) に関する研究が盛んに行われてます。動物実験ではアガリクスによって移植されたガン細胞が消失したという報告もあります。ただしヒトを対象にした臨床試験は未だ行われておらず体験談や症例報告が示されているに過ぎません。

この他アガリクスにはガン細胞の増殖を直接抑制する成分であるステロイド類が含まれてます。また同じような効果を示す新しいステロイド類も発見されてます。この新しいステロイド類は脂溶性成分のエリゴステロールから生成されたと考えられてます。エリゴステロールはアガリクスのみならず多くのきのこ類に含まれており脚光を浴びてます。

またアガリクスに沢山含まれる食物繊維不飽和脂肪酸は悪玉コレステロールの受容体 ( レセプター ) の構造と機能を活性化させます。これにより悪玉コレステロールは血中に放出されず体外に排出されます。

アガリクス子実体とアガリクス菌糸体どれがいいの？

メシマコブと同様にアガリクスについてもまた子実体と菌糸体のどちらが優れているかという事は現段階では結論が出ていないのが現状です。

また子実体か菌糸体かによって製法も変わってきます。例えば酵素処理であれば菌糸体が使用されます。乳酸菌による醗酵処理であれば通常、子実体が使用されます。栽培方法も子実体であれば露地栽培のものが菌糸体であればタンク培養のものがよいと言われてます。

アガリクスは医薬品ではなく健康食品なので一日目安量は定められていません。アガリクスを栄養補助食品 ( サプリメント ) として摂取する時は製品表示を確認して下さい。
栄養補助食品としてのアガリクスは顆粒錠剤カプセルパウダー子実体 ( きのこ ) の乾燥品等が市販されてます。通常、アガリクスの菌株培地製法等によって品質に差があります。中には健康被害が出ている製品もあります。サプリメントに詳しい医師や薬剤師又はサプリメントアドバイザーに相談するか信頼できるメーカーの製品を購入するようにしましょう。治療中の疾患がある時は必ず医師の指示に従って下さい。

アガリクス？カワリハラタケ？ヒメマツタケ？

アガリクス茸という固有のきのこは存在しません―――。よく目にする一文です。

ハラタケ科ハラタケ属アガリクスという名称を持つきのこは確かに存在しません。しかし 通常、アガリクス又はアガリクス茸と言えばアガリクス・ブラゼイ・ムリルを指してます。

［アガリクス・ブラゼイ・ムリル］は学術名である為長く言いにくく、また覚えにくいです。その為にアガリクス又はアガリクス茸と省略しているに過ぎないでしょう。

一方姫マツタケ ( ヒメマツタケ ) もよく目にする名称です。

まず岩出亥之助博士がアガリクス ( の種菌 ) を入手したときアガリクス・ブラゼイ・ムリルという学術名はまだ固定されていませんでした。これが 1965 年の事です。アガリクス・ブラゼイ・ムリルという学術名が固定されたのが 1967 年の事。岩出博士がアガリクス・ブラゼイ・ムリルという学術名を知ったのは当然 1967 年以降の事になります。この時岩出博士はすでにアガリクスの人工栽培の研究に取り組んでおり研究所ではカワリハラタケ ( ハラタケの新種という意味 ) と呼ばれていたと言われてます。

次に岩出博士がアガリクスの人工栽培に成功したのが 1975 年これが市場に出荷されたのが 1978 年です。この時ヒメマツタケという名称が使用されました。また栽培されていたアガリクスの種菌 ( 菌株 ) は岩出 101 株と命名されてます。そして1982 年の日本菌学会誌ではアガリクス・ブラゼイ・ムリルは 和名カワリハラタケ 実用名ヒメマツタケ と記載され 1987 年版原色日本新菌類図鑑では 和名ヒメマツタケ 別名カワリハラタケ と収載されました。アガリクスの菌株は数種類ありますが岩出 101 株をもとに栽培されたアガリクスは間違いなくヒメマツタケです。

実にややこしい話です。

消費者としてはでは岩出 101 株をもとに栽培されたヒメマツタケはアガリクス・ブラゼイ・ムリルと全く等しいのか又は岩出 101 株以外の菌株をもとに栽培されたアガリクスはヒメマツタケと全く等しいかという疑問が当然起こります。その判断は専門家にお任せするしかないでしょう。
