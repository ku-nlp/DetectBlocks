胃がんとは胃の内部に悪性（がん）細胞が認められる病気です。

胃は上腹部に存在するアルファベットのJの形をした臓器で、飲食した食物中の栄養素（ビタミン、ミネラル、炭水化物、脂肪、蛋白質、水分）を処理し、老廃物を体外に排泄するのに役立つ消化器系の一部です。食物は食道と呼ばれる空洞状の筋肉の管を通って口（咽喉）から胃まで運ばれます。胃を通過した後に、部分的に消化された食物は小腸を通って次に大腸まで進みます。

胃壁は粘膜層（最内層）、筋肉層（中間層）および漿膜層（最外層）の3つの組織層から構成されています。胃がんは粘膜層の内側にある細胞から生じますが、成長とともに外層まで拡がります。

胃の間質性腫瘍は支持組織である結合組織から生じ、胃がんに対する治療法とは異なります。詳しい情報についてはPDQの成人軟部組織肉腫＊の治療についての項目を参照してください。

（注）＊の項目はがんinfo.の項目には含まれていません。

年齢、食事、胃の病気により胃がんの発生リスクに影響が出ます。

リスク因子には下記のようなものがあります：
  • ヘリコバクターピロリという細菌に胃が感染している。
  • 慢性胃炎（胃の炎症）がある。
  • 高齢者。
  • 男性。
  • 塩分が多い食物、燻製食品または保存状態の悪い食物を多く食べ、果物や野菜をあまり食べない。
  • 悪性貧血。
  • 喫煙する。
  • 腸上皮化生がある。
  • 家族性大腸腺腫症または胃ポリープがある。
  • 母親、父親、姉妹または兄弟に胃がんの人がいる。

胃がんを疑う症状としては、消化不良、胃部不快感、疼痛などがあります。

これらの症状は胃がんによってみられることがあります。他の状況によっても同じ症状がみられることがあります。

胃がんの早期には、以下の症状がみられることがあります：
  • 消化不良や胃の不快感。
  • 食後の膨満感。
  • 軽度の悪心。
  • 食欲低下。
  • 胸焼け。
胃がんの病期がさらに進行すると、以下の症状がみられることがあります：
  • 血便。
  • 嘔吐。
  • 原因不明の体重減少。
  • 胃痛。
  • 黄疸（皮膚や白目が黄色くなる）。
  • 腹水（腹腔内に液体が貯まる）。
  • 嚥下困難。
これらの症状がひとつでもみられた際には医師の診察を勧めます：

胃がんを発見し、診断するために、胃と食道を調べるための検査が用いられます。

以下の試験や手法が用いられます：
理学的所見および既往歴：
全身を調べて、一般的健康状態をチェックするとともに腫瘤または増殖など疾患徴候をチェックします。また患者の生活習慣や過去の疾患、治療の病歴についても調べます。
血液化学的検査：
身体中の器官、組織によって放出されるある物質の量を測定するために、血液サンプルを調べる手法です。ある物質の量が異常（正常より多いまたは少ない）である場合、それをつくる器官、組織における疾患の徴候である可能性があります。
全血球数算定検査：
血液サンプルを採取し、下記の項目を調べる手法です：
  • 赤血球、白血球および血小板数。
  • 赤血球中のヘモグロビン（酸素を運ぶ蛋白質）量。
  • サンプルにおける赤血球の構成比率。
上部内視鏡検査：
異常部位を調べるために食道、胃、十二指腸（小腸の最初の部分）の内部を観察する手法です。内視鏡（細く、ライトの付いた管）を口から喉を通して食道まで挿入します。
便潜血検査：
大便（固形排泄物）中に顕微鏡でのみ観察できるような血液があるかどうかを調べる検査です。検便用の小サンプルは特定のカードに入れて、検査のために医師または臨床検査室に戻されます。
バリウム嚥下：
食道と胃の連続したX線像です。バリウム（銀白色の金属化合物）を含む溶液を飲みます。液体は食道と胃を覆い、X線が撮影されます。この手法は上部消化管撮影とも呼ばれます。
生検：
細胞または組織を採取して顕微鏡下で観察し、がんの徴候を調べます。通常胃の生検は内視鏡検査中に行われます。
CTスキャン（CATスキャン）：
いろいろな角度から体内の詳細な像を連続的に撮影します。像はX線撮影装置と連動したコンピュータにより作られます。造影剤を静脈内に注入または飲みこむと、臓器や組織がよりはっきり示されます。この方法はまたコンピュータ断層撮影法、またはコンピュータ体軸断層撮影法（CAT）とも呼ばれています。

諸条件により治療法の選択や予後（治癒の可能性）が変わります。

治療法の選択と予後（治癒の可能性）はがんの病期・程度（胃のみかリンパ節まで拡がっているか、または体の他の部分まで拡がっているか）と患者の全身状態によって異なります。

胃がんは非常に早期に発見された場合、、治癒の可能性が高いです。しかし診断時に既に胃がんの病期が進行している場合があります。後期の病期では、胃がんに対して治療は行えますが、治癒する可能性は稀です。治療を改善するために行われている臨床試験の1つに参加することも考慮されます。
現在行われている臨床試験についての情報はNCI Web siteにアクセスすれば入手できます。

胃がんと診断された後、がん細胞が胃内にとどまっているか、それとも体の他の部分まで拡がっているかを調べる目的で諸検査を行います。

がんが胃内にとどまっているか、それとも体の他の部分まで拡がっているかを調べるために行われる検査を病期診断といいます。病期診断のために行われた検査から得られた情報を基に疾患の病期が決定されます。最善の治療計画を立てるために病期を把握することは重要です。

病期診断に用いられる検査には次のようなものがあります：
βHCG（ベータヒト絨毛性ゴナドトロピン）、CA-125およびCEA（がん胎児性抗原）の測定：
血液中のβHCG、CA-125およびCEA濃度を測定する検査です。これらの物質はがん細胞と正常細胞の両方から血流中に放出されます。正常量よりも多く認められる場合、胃がんまたは他の状況の徴候である可能性があります。
胸部X線検査：
胸部に簡単なX線照射を行い、胸部とその内部構造のＸ線像です。Ｘ線とは体内を通過してフィルム上まで達し、体内を撮影することができるエネルギービームの一種です。
超音波内視鏡検査（EUS）：
内視鏡（細く、ライトの付いた管）を体内に挿入する検査です。内視鏡は体内の組織や臓器に高エネルギー超音波を反射させて、エコーをつくるために用いられます。エコーはソノグラムと呼ばれる体内組織の像をつくります。この検査は内視鏡超音波検査とも呼ばれます。
CTスキャン（CATスキャン）：
いろいろな角度から体内の詳細な像を連続的に撮影します。像はX線撮影装置と連動したコンピュータにより作られます。造影剤を静脈内に注入または飲みこむと、臓器や組織がよりはっきり示されます。この方法はまたコンピュータ断層撮影法、またはコンピュータ体軸断層撮影法（CAT）とも呼ばれています。
腹腔鏡検査：
異常部位を調べるために腹腔内の臓器を観察する外科的手法です。腹壁に切開部（切れ目）をつくり、腹腔鏡（細く、ライトの付いた管）を腹部に挿入します。器官の除去や生検のために組織サンプルを採取することもあります。
PETスキャン（陽電子放射断層撮影）：
体内にある悪性腫瘍細胞をみつけるための手法です。少量の放射性核種グルコース（糖）を静脈内に注入します。PETスキャナーが体の周囲を回転してグルコースが体内で利用されている部分の像を撮影します。悪性がん細胞は正常細胞よりも活発で、グルコースをより多く吸収することから、像はより明るく示されます。

胃がんの病期は以下の通りです：

０期（上皮内がん）
０期（上皮内がん）では、がんは胃壁の粘膜層（最内層）の内側にのみ認められます。
I期
I期の胃がんはがんがどこまで拡がったかにより、IA期とIB期に分けられます。
IA期：
がんは完全に胃壁の粘膜層（最内層）全体に拡がっています。
IB期：
がんは以下の部分まで拡がっています；
  • がんは完全に胃壁の粘膜層（最内層）全体に拡がっており、さらに腫瘍付近の6カ所までのリンパ節にも認められる；
  • 胃壁の筋肉層（中間層）にまで拡がっている。
II期
II期の胃がんでは、がんは以下の部分まで拡がっています；
  • がんは完全に胃壁の粘膜層（最内層）全体に拡がっており、さらに腫瘍付近の7〜15カ所のリンパ節にも認められる。
  • がんは胃壁の筋肉層（中間層）にまで拡がっており、さらに腫瘍付近の6カ所までのリンパ節にも認められる。
  • がんは胃壁の漿膜層（最外層）まで拡がっているが、リンパ節または他の臓器までは拡がっていない。
III期
III期の胃がんでは、がんがどこまで拡がっているかによってIIIA期とIIIB期に分けられます。
IIIA期：
がんは以下の部分まで拡がっています；
  • 胃壁の筋肉層（中間層）まで拡がっており、さらに腫瘍付近の7〜15カ所のリンパ節に認められる；
  • 胃壁の漿膜層（最外層）まで拡がっており、さらに腫瘍付近の1〜6カ所のリンパ節に認められる；
  • 胃に隣接した臓器まで拡がっているが、リンパ節または体の他の部分までは拡がっていない。
IIIB期：
がんは胃壁の漿膜層（最外層）にまで拡がっており、さらに腫瘍付近の7〜15カ所のリンパ節に認められる。
IV期
IV期では、がんは以下の部分まで拡がっています：
  • 胃に隣接した臓器と1カ所以上のリンパ節まで拡がっている；
  • 15カ所以上のリンパ節まで拡がっている；
  • 体の他の部分まで拡がっている。

再発性胃がん

再現性胃がんは治療後に再び生じた（再燃）がんのことをいいます。がんは胃または肝臓やリンパ節など体の他の部分に再発することがあります。

このセクションの要点
  • 胃がん患者に対して様々なタイプの治療法があります。
  • 標準的治療法として以下の4種類が用いられます：
    • 手術
    • 化学療法
    • 放射線療法
    • 化学放射線療法
  • その他の治療法は現在、臨床試験で有効性を検討中です。これらには以下のようなものがあります。
    • 生物学的療法

胃がん患者に対して様々なタイプの治療法があります。

胃がん患者に対しては様々なタイプの治療法が適用されます。標準的治療法（現在用いられている治療法）もあれば、臨床試験において検証されているものもあります。治療を始める前に、臨床試験に参加したいと考える患者がいるかもしれません。治療法についての臨床試験は、現在行われている治療法の改善やがん患者の新しい治療法に対する情報を得るために行われるものです。現時点で標準的とされている治療法よりも新しい治療法の方が良いと示された場合、今度は新しい治療法が標準的治療法になる可能性があります。
臨床試験はアメリカ国内のいろいろなところで実施されています。実施されている臨床試験についての情報はインターネットでNCI Web siteにアクセスすれば、入手できます。患者とその家族、そして医療側が一体となって治療法を確定することががんの最善の治療法を選ぶ理想的な姿です。

標準的治療法として以下の4種類が用いられます：

手術
手術は全ての病期の胃がんに対して一般的治療法です。次のようなタイプの手術が用いられます：
胃亜全摘術：
がんを含む胃の部分、隣接したリンパ節、腫瘍に隣接した他の組織や臓器を除去します。脾臓を摘出することもあります。脾臓は上腹部にある臓器で、血液をろ過し、老化した血液細胞を排除します。
胃全摘術：
胃全体、隣接したリンパ節、食道、小腸および腫瘍に隣接した他の組織の一部を摘出します。脾臓を摘出することもあります。以前と同様に食物を食べたり、飲み込んだりできるように、食道と小腸をつなぎます。
腫瘍が胃の開口を閉鎖しており、標準的手術によりがんを完全に摘出できない場合、以下のような処置が用いられることがあります：
管腔内ステント留置術：
（動脈や食道などの）管腔の開口を維持するためにステント（細く、拡張可能な管）を挿入する手法です。胃の開口を閉鎖する腫瘍に対しては、手術を行って患者が普通に食物を食べられるように食道から胃までステントを留置することがあります。
内視鏡的レーザー手術：
レーザーを備えた内視鏡（細く、ライトの付いた管）を体内に挿入する処置です。レーザーはナイフとして用いることもできる強力な光線です。
電気焼灼術：
電流を用いて熱を発生させる手法です。時に病変部を摘出したり、止血するために用いられることもあります。
化学療法
化学療法は、薬剤を用いてがん細胞を殺すかまたは細胞分裂を停止させることでがん細胞の増殖を停止させるがん治療のことです。口から服用したり、筋肉や静脈内に注入する化学療法では、薬剤は血流を通って全身のがん細胞に影響します（全身療法）。脊柱、腹部、臓器などの体腔に薬剤を直接注入する化学療法では、薬剤は主にこれらの領域中にあるがん細胞に影響します。化学療法はがんの種類や病期によって異なります。
放射線療法
放射線療法は高エネルギーX線やその他の種類の放射線を用いてがん細胞を殺すがん治療のことです。放射線療法には2つのタイプがあります。体外照射は体外の機械を用いてがんに放射線を照射する治療法です。体内照射は放射性物質を密封した針、シーズ、ワイヤ、カテーテルをがんの内部またはその近くに直接留置して、がんに放射線を照射する治療法です。成人非ホジキンリンパ腫に対しては外照射が用いられます。放射線療法の方法はがんの種類や病期によって異なります。
化学放射線療法
化学放射線療法は、化学療法と放射線療法を併用し、両方の効果を増強させるものです。治癒の可能性を高めるために手術後に行われる化学放射線療法はアジュバント療法と呼ばれます。また手術前に行われる場合にはネオアジュバント療法と呼ばれます。

その他の治療法は現在、臨床試験で有効性を検討中です。これらには以下のようなものがあります。

生物学的療法
生物学的療法は、患者自身のがんと闘う免疫機構を活発にする治療法です。自らの体内でつくられる物質や実験室で作成された物質を用い、患者自身の病気に対するもともとの抵抗力を高め、方向づけしたり、回復させたりします。このがん治療法はまたバイオセラピー、免疫療法とも呼ばれます。
このまとめのセクションでは、現在臨床試験を行っている治療法について個々に触れますが、最新の臨床試験をすべて網羅できていない可能性があります。アメリカで実施されている臨床試験についての情報はインターネットでNCI Web siteにアクセスすれば、入手できます。

IV期胃がん

遠隔臓器にまで拡がっていないIV期胃がんの治療法には以下のようなものがあります：
  • 手術（胃全摘術）後の化学放射線療法。
  • 手術前化学放射線療法の臨床試験。
遠隔臓器まで拡がっているIV期胃がんの治療法には以下のようなものがあります：
  • 症状を緩和しQOLを改善するための緩和療法としての化学療法。
  • 症状を緩和しQOLを改善するための緩和療法としての内視鏡的レーザー手術または管腔内ステント留置術。
  • 止血、鎮痛または胃の開口を閉鎖している腫瘍を縮小するための緩和療法としての放射線療法。
  • 止血または胃の開口を閉鎖している腫瘍を縮小するための緩和療法としての手術。

再発性胃がんに対する治療法

再発性胃がんに対する治療法には以下のようなものがあります：
  • 症状を緩和しQOLを改善するための緩和療法としての化学療法。
  • 症状を緩和しQOLを改善するための緩和療法としての内視鏡的レーザー手術または電気焼灼術。
  • 止血、鎮痛または胃を閉鎖している腫瘍を縮小するための緩和療法としての放射線療法。
  • 新規抗がん剤または生物学的療法の臨床試験。
このまとめのセクションでは、現在臨床試験を行っている治療法について個々に触れますが、最新の臨床試験をすべて網羅できていない可能性があります。アメリカで実施されている臨床試験についての情報はインターネットでNCI Web siteにアクセスすれば、入手できます。
